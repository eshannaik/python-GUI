# Contact-Book

#### The main purpose of this project is to make a GUI for a Contact Book.
#### This is done using PyQt5.
#### To install PyQt5 -
    pip install PyQt5
#### To run the program open cmd and type "python rpcontacts.py".

#### The libraries used are -
    PyQt5
    sys
